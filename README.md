# freelancer-portfolio

> Client deliverable bundle — Carolina design system + portfolio assets

A working folder for a freelance UI deliverable named "Carolina". Contains the full design-system specification (`DESIGN.md`, `skill.md`), reference screenshots of the contact/about page sections, scrolling-image component reference, and a complete inner portfolio project (`girish-portfolio/`) with its own README, agent docs, and append notes.

🔗 **Live repo:** <https://github.com/girishlade111/freelancer-portfolio>

## ✨ Features

- Carolina brand design system — mission, tokens, component rules
- Annotated contact-page screenshots (header, about, FAQ, footer)
- Scrolling image component reference assets
- Embedded `girish-portfolio/` sub-project with its own README + AGENTS.md
- Machine-readable `skill.md` for design-system agents

## 🛠️ Tech stack

Markdown • Design tokens • Reference screenshots • Agent-driven documentation

## 🚀 Getting started

```bash
# This repo is a reference deliverable — clone to browse
git clone https://github.com/girishlade111/freelancer-portfolio.git
# Read DESIGN.md and skill.md for the design system
```
